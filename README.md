# PlayerPath

**A comprehensive baseball/softball performance tracking app for athletes and coaches.**

---

## ğŸ† **What is PlayerPath?**

PlayerPath is an iOS app that helps baseball and softball athletes:
- **Track performance statistics** (batting average, hits, runs, RBIs, etc.)
- **Record and analyze game videos** with play-by-play results
- **Share videos with coaches** for personalized feedback
- **Export statistics** for recruiting and record-keeping
- **Manage seasons, games, and practices**
